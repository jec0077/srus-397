Before Running This Directory, Install all the required libraries using command:
    `pip install -r .\requirements.txt`

If Necessary, Use Python Virtual Environment using command:
    `.\venv\Scripts\activate`

To Exit Virtual Environment, use command:
    `deactivate`

If You Want to Commit Any Changes to the Main Branch
    `git status`
    `git add -a`
    `git commit -m "message"`
    `git push -u origin main`
